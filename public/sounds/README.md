# Sound Files

Add the following MP3 files to this directory:

1. **spin.mp3** - A spinning/slot machine sound effect (3-4 seconds)
2. **win.mp3** - A celebratory/winning sound effect (1-2 seconds)

You can get free sound effects from:
- https://freesound.org
- https://zapsplat.com
- https://soundbible.com

Or use these placeholder sounds:
- Spin sound: Any slot machine or spinning wheel sound
- Win sound: Any celebratory chime or fanfare
